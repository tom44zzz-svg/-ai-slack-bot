# -ai-slack-bot

ネタ（スプレッドシート）を入力すると、Canva の投稿フォーマットを選び、
テキスト原案とビジュアル指示を返す Slack bot。

```
ネタ（スプレッドシート）
      ↓
フォーマットを選ぶ or おすすめされる
      ↓
テキスト原案 ＋ ビジュアル指示
```

## 現在のフェーズ：フォーマット仕様書づくり

bot 本体の実装の前段として、過去の Canva 投稿（1〜3 月分）を棚卸しし、
フォーマットの「型」を言語化する。仕様書は人間・AI 双方から参照できる形で残す。

各フォーマットは以下 4 観点で定義する：

1. 構成（スライド枚数・各ページの役割・テキスト配置）
2. 文字数上限（タイトル・本文・キャプション別）
3. 図解／ビジュアルの種類
4. 向いているテーマ・ネタの特性

## ディレクトリ構成

```
.
├── README.md                 # このファイル
├── docs/
│   ├── format-spec.md        # 人間向けフォーマット仕様書
│   └── format-catalog.md     # 各フォーマットの詳細カタログ
└── data/
    ├── formats.yaml          # 機械可読なフォーマット定義
    └── source-posts.yaml     # 分析元となった Canva 投稿のメタ情報
```

## 今セッションの成果物

- `data/formats.yaml` ：フォーマット型の辞書（bot 実装時にそのまま使う）
- `data/source-posts.yaml` ：分析元となった過去投稿一覧
- `docs/format-spec.md` ：仕様書本体
- `docs/format-catalog.md` ：フォーマット別の詳細ページ

- LLM プロンプト実装
