# 図解ギャラリー

`data/diagram-gallery.yaml` の人間可読版。
Web アプリではこの一覧を画像サムネイルでビジュアル表示し、クリック選択可能にする。

---

## 2値比較系

### `compare_before_after` Before/After 2 点対比
```
┌────┐          ┌────┐
│ 前 │   ─→    │ 後 │
└────┘          └────┘
```
**用途**：施策前後・状態変化・金額変化
**例**：`post_shinsa_5` の「融資希望額を減らす」（5,000 万円 → 4,000 万円）

---

### `compare_vs_2col` 2 列 VS 比較
```
┌────────┐  ┌────────┐
│ A 側   │  │ B 側   │
│ 特徴   │  │ 特徴   │
└────────┘  └────────┘
```
**用途**：A と B を 3 要素程度で列挙

---

### `compare_pros_cons` メリット / デメリット
```
◎ メリット       × デメリット
・項目            ・項目
```

---

### `compare_do_dont` 対策してる人 / してない人
```
🙆 している人    🙅 していない人
特徴             特徴
```

---

## N値比較系

### `compare_table_row` 複数行比較表
```
│ 軸     │ A │ B │ C │
│ 軸 1   │ ○ │ × │ △ │
│ 軸 2   │ ○ │ ○ │ × │
```
**用途**：3〜4 対象 × 複数軸
**例**：`post_shinsa_5` の「通りやすい人 vs 通りにくい人」

---

### `compare_3_column` 3 列カード比較
```
┌──────┐  ┌──────┐  ┌──────┐
│ 現物 │  │ 換価 │  │ 共有 │
└──────┘  └──────┘  └──────┘
```
**用途**：3 選択肢の違いを視覚化
**例**：`post_souzoku_bunkatsu` の「代償分割以外の分け方」

---

### `icons_row_3` 3 アイコン並列
```
🔺   💰   📋
 A    B    C
```
**用途**：3 つの選択肢を短ラベルで

---

### `cards_2x2` 2×2 グリッド（4 選択肢）
```
┌──┐  ┌──┐
│ 1│  │ 2│
└──┘  └──┘
┌──┐  ┌──┐
│ 3│  │ 4│
└──┘  └──┘
```

---

## 階層・関係系

### `tree_relation` ツリー関係図
親子・分配・分岐を示すツリー

### `persona_pass_fail` 人物 → 丸抜き選択（○×）
```
👤 → A社 B社 C社
     [×] [×] [○]
```
**例**：`post_shinsa_5` の「別の金融機関に相談する」

## 強調系

### `number_highlight` 大数字強調
```
┌─────┐
│ 80% │
└─────┘
～の人が知らない
```
**要件**：制作ルール 3（国・企業データ限定）

**例**：`post_shinsa_5` の「『〇〇のに…』『〇〇なのに…』」

```

```

```

│ ⚠ 注意       │
└──────────────┘
```
**用途**：失敗・NG 型で警告を明示
